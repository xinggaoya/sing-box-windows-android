# Sing Box Windows Android - v1.0.0 初始发布

## 🎉 首个正式版本发布

Sing Box Windows Android 是一个基于 sing-box 核心的现代化 Android VPN 客户端应用，提供简洁、高效的网络代理管理功能。

### ✨ 主要特性

- 🚀 **高性能 VPN 服务**：基于 sing-box 核心，支持多种代理协议
- 📱 **现代化界面**：采用 Jetpack Compose 构建的 Material 3 设计风格界面
- 🔗 **智能订阅管理**：支持添加、编辑和管理多个订阅链接
- ⚡ **实时节点测速**：自动测试节点延迟，帮助选择最优节点
- 📊 **流量监控统计**：实时显示网络连接和流量使用情况
- 🎯 **自动节点分组**：智能组织和管理节点分组
- 🔧 **多架构支持**：支持 ARM64 和 ARMv7 架构设备
- 🌐 **协议兼容性**：完整支持 VLESS、Shadowsocks 等主流代理协议

### 📦 下载说明

本项目支持多种架构的 APK 文件，请根据您的设备架构选择合适的版本：

- `sing-box-windows-android-v1.0.0-arm64-v8a.apk`：适用于 64 位 ARM 架构设备（**推荐**）
- `sing-box-windows-android-v1.0.0-armeabi-v7a.apk`：适用于 32 位 ARM 架构设备

#### 如何选择正确的版本？

1. **大部分现代设备**请选择 `arm64-v8a` 版本
2. **较老的设备**可以选择 `armeabi-v7a` 版本
3. 如果不确定，可以尝试安装 `arm64-v8a` 版本，若不兼容再使用 `armeabi-v7a`

### 📋 系统要求

- **Android 版本**：Android 10 (API Level 29) 或更高版本
- **权限要求**：VPN 权限和网络访问权限
- **存储空间**：至少 100MB 可用空间

### 🚀 快速开始

1. 下载对应架构的 APK 文件
2. 在 Android 设备上安装应用
3. 授予必要的 VPN 和通知权限
4. 添加您的订阅链接或手动配置
5. 选择节点并连接

### 🔧 技术规格

- **开发语言**：Kotlin + Jetpack Compose
- **核心库**：libbox (sing-box)
- **构建工具**：Gradle with Kotlin DSL
- **UI 框架**：Material Design 3
- **最低 SDK**：API 29
- **目标 SDK**：API 36
